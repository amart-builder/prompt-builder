# Task Type Taxonomy

Nine types, one framework each. Classify the request into exactly one.

| # | Type | Framework | Best For |
|---|---|---|---|
| 1 | CODING | Specification-Driven Development (SDD) | Building features, scripts, tools |
| 2 | RESEARCH | Structured Analytical Framework (SAF) | Investigation, comparison, analysis |
| 3 | WRITING | Voice-Matched Composition (VMC) | Content in the user's voice, emails, posts |
| 4 | PLANNING | Decision Architecture (DA) | Strategy, roadmaps, prioritization |
| 5 | DATA | Pipeline Specification (PS) | ETL, transforms, data analysis |
| 6 | DEBUGGING | Diagnostic Protocol (DP) | Fixing broken things, root cause analysis |
| 7 | CREATIVE | Creative Brief (CB) | Design, branding, visual concepts |
| 8 | OUTREACH | Relationship-Context Composition (RCC) | Cold outreach, LP comms, introductions |
| 9 | ORCHESTRATION | Context Bundle Protocol (CBP) | Sub-agent tasks, cron payloads, multi-step workflows |

Full framework templates: `references/framework-templates.md`
